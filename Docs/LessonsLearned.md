# 採用管理システム開発で学んだこと

## はじめに

採用管理システムの開発を通じて、単にシステムを作るだけでなく、業務を理解することの重要性を学びました。

## 学んだこと

### ヒアリングの重要性

開発を始める前に採用担当者へヒアリングを行い、実際の業務フローや課題を整理しました。

### 業務フローの可視化

採用プロセスをフロー化することで、どこに課題があるのかを把握しやすくなりました。

### KPI設計の重要性

採用率や入社率などの指標を定義することで、採用活動を数値で把握できるようになりました。

### システムは業務改善の手段

重要なのはシステムそのものではなく、業務改善を実現することだと学びました。

## 今後に活かしたいこと

今後も業務課題を整理し、利用者の視点を意識したシステム開発を行っていきたいと考えています。
