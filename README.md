# 採用管理システム

## 概要

採用活動における応募者管理および採用状況分析を効率化するために作成したシステムです。

## 主な機能

- 応募者情報管理
- 面接結果管理
- 採用状況集計
- 採用率分析
- グラフ表示

## 使用技術

- Microsoft Access
- VBA
- Microsoft Excel

## 開発背景

採用担当者から採用活動における課題や採用の流れなどをヒアリングし、採用プロセスをフロー化しながら改善点を整理しました。

応募者情報や採用率を効率的に管理し、手作業による集計や情報管理の負荷を軽減することを目的として開発しました。

また、採用状況を可視化することで、より効率的な採用活動を支援することを目指しました。

## 解決した課題

- 応募者情報の一元管理
- 選考状況の可視化
- 採用率の分析
- 集計作業の効率化
- 仲介エージェント会社ごとの採用率および紹介件数管理

## 工夫した点

- 採用率をグラフ表示し、採用状況を直感的に把握できるようにした
- 現在の運用実態に合わせ、採用率は申込日を基準に算出し、入社率の集計時には入社日を基準とするなど、実際の業務に即した指標設計を行った
- 入力作業を簡素化し、運用負荷を軽減した
- 集計処理を自動化し、手作業によるミスや作業時間を削減した
